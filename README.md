# MendianPilot

AI 店铺获客自动驾驶原型。

## 产品边界

- 面向国内下沉市场小店。
- 商家自助上传店铺/产品/案例照片。
- AI 自动生成内容、发布排期、评论钩子、到店接待话术和线索看板。
- 不做人工私域、不做代运营、不承诺涨粉成交。
- 平台发布优先走官方授权能力；没有官方能力时只做提醒、复制和下载。

## MVP 链路

1. 选择行业和月卡。
2. 上传素材照片。
3. 填写店名、客户、地址、优惠。
4. 生成本周获客包。
5. 查看可发布内容、排期、自动接待和线索。
